# Configuration

Configuration templates and defaults for Persona.

## Structure

```
config/
├── vendors/          # LLM provider configurations
│   ├── openai.yaml
│   ├── anthropic.yaml
│   └── gemini.yaml
├── models/           # Model definitions
│   ├── openai/
│   ├── anthropic/
│   └── gemini/
├── prompts/          # Prompt templates
├── workflows/        # Workflow definitions
└── synthetic_data/   # Sample data for testing
```
