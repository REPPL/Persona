# Developer Persona Template

Use this template to generate developer-focused personas.

## Persona Structure

Generate a persona with the following sections:

### Basic Information
- Name (fictional)
- Role/Title
- Years of Experience
- Primary Skills

### Daily Workflow
What does a typical day look like for this person?

### Goals
What are they trying to achieve in their role?

### Pain Points
What frustrates them or slows them down?

### Tools & Technologies
What tools do they use regularly?

### Motivations
What drives them in their work?

### Key Quotes
Include 2-3 quotes from the source data that exemplify this persona.

## Usage

```bash
persona generate --from data/interviews.csv --template templates/developer-persona.md
```
