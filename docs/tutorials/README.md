# Tutorials

Learning-oriented guides that help you get started with Persona.

## Available Tutorials

| Tutorial | Description | Level |
|----------|-------------|-------|
| [Getting Started](01-getting-started.md) | Your first persona generation | Beginner |

## What You'll Learn

Tutorials teach you the fundamentals through hands-on examples:

1. **Installation** - Set up Persona on your system
2. **Configuration** - Connect to your LLM provider
3. **Experiments** - Organise your research projects
4. **Generation** - Create your first personas
5. **Results** - Understand and use the output

## Prerequisites

- Python 3.12 or higher
- API key for at least one LLM provider (OpenAI, Anthropic, or Gemini)
- Basic command-line knowledge
