# Milestone: v0.1.0 - Foundation

The foundation release establishing basic persona generation capability.

## Goal

Deliver a working persona generation system that can load data, send to an LLM, and produce structured persona output. Supports OpenAI, Anthropic, and Gemini.

## Theme

**"Data to Personas"** - The essential generation loop.

## CLI Commands

| Command | Description |
|---------|-------------|
| `persona check` | Verify installation and provider configuration |
| `persona create experiment` | Create new experiment |
| `persona list experiments` | List existing experiments |
| `persona show experiment <name>` | Show experiment details |
| `persona delete experiment <name>` | Delete experiment |
| `persona generate --from <source>` | Generate personas |
