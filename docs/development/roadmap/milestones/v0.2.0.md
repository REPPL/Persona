# Milestone: v0.2.0 - Multi-Step Workflows

Sequential workflows with intermediate outputs.

## Goal

Enable multi-step persona generation workflows where each step builds on the previous, with intermediate results saved for inspection.

## Theme

**"Extract → Consolidate → Validate"** - Research-grade workflows.

## Features Included (9)

| F-020 | Workflow orchestration | Workflow |

| F-023 | Output chaining between steps | Workflow |
| F-024 | Intermediate output saving | Output |
| F-025 | Persona consolidation step | Generation |
| F-026 | Persona validation step | Generation |
| F-027 | Prompt template validation | Prompts |

## Success Criteria

- [ ] Multi-step workflows execute in sequence
- [ ] Intermediate outputs saved to `raw/` folder
- [ ] Each step can use different model
- [ ] Output from step N available to step N+1
- [ ] Test coverage ≥ 80%
