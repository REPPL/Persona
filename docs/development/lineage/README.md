## Key Differences from PersonaZero

| Aspect | PersonaZero | Persona |
|--------|-------------|---------|
| Specification | Ad-hoc | Hybrid (Use Cases → Features → Tutorials) |
| Roadmap | Version-centric | Feature-centric |
| Testing | Deferred | Alongside implementation |
| Documentation | Reactive | Documentation-as-you-go |
| Versioning | Alpha/beta tags | Minor releases only |
