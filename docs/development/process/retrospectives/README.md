# Milestone Retrospectives

Post-milestone analysis and lessons learned.

## Purpose

Retrospectives document:
- What happened vs. what was planned
- Manual interventions required
- Documentation drift discovered
- Process improvements for next milestone

## Format

```markdown
# Retrospective: vX.Y.0

**Date:** YYYY-MM-DD

## Summary
[Brief overview of the milestone]

## What Went Well
- [Positive 1]
- [Positive 2]

## What Needed Improvement
- [Issue 1] → [How we'll address it]
- [Issue 2] → [How we'll address it]

## Manual Interventions
| Intervention | Why Needed | Automatable? |
|--------------|------------|--------------|
| ... | ... | ... |

## Documentation Drift
| Document | Issue | Resolution |
|----------|-------|------------|
| ... | ... | ... |

## Action Items
- [ ] Action for next milestone
```

*Retrospectives will be added after each milestone.*
