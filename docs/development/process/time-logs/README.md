# Time Logs

Time tracking for development work.

## Purpose

Time logs track:
- Actual hours spent per feature/milestone
- Comparison to estimates
- Insights for future planning

## Format

```markdown
# Time Log: [Feature/Milestone]

## Summary
- **Estimated:** X hours
- **Actual:** Y hours
- **Variance:** Z%

## Breakdown
| Task | Estimated | Actual | Notes |
|------|-----------|--------|-------|
| ... | ... | ... | ... |

## Lessons
[What we learned about estimation]
```
