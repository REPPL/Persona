## Architecture Principles

1. **Clean Separation** - Core logic independent of interfaces
2. **Configuration-Driven** - Behaviour controlled by YAML, not code
3. **Provider-Agnostic** - LLM providers behind unified interface
4. **Testable** - All components mockable and testable
