## What Belongs Here

Vision documents define:
- **Goals** - What we want to achieve
- **Non-goals** - What we explicitly won't do
- **Principles** - Design philosophy
- **Success criteria** - How we measure achievement

## What Doesn't Belong Here

- Feature specifications → [Features](../../roadmap/features/)
- Implementation details → [Implementation](../../implementation/)
- Release schedules → [Milestones](../../roadmap/milestones/)
