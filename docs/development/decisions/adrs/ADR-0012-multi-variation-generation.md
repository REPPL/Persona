# ADR-0012: Multi-Variation Generation

## Status

Accepted

## Context

Users want to:
- Compare different settings
- Find optimal quality/cost balance
- Test complexity levels
- Generate comparison reports

## Decision

Support multi-variation generation (v0.3.0):
- Complexity levels: simple, moderate, complex
- Detail levels: minimal, detailed
- All combinations in single run
- Automatic comparison analysis

## Consequences

**Positive:**
- Easy A/B testing
- Quality optimisation
- Single command for multiple variations
- Research-friendly

**Negative:**
- Multiplied costs
- Longer execution time
- More complex output structure
