# AI Prompts

This directory contains prompts for AI image and content generation.

## Contents

| File | Purpose |

## Usage

These prompts are designed for use with AI image generation tools (Gemini, DALL-E, Midjourney, etc.). Copy the prompt text and paste into your preferred tool.

## What Belongs Here

- Image generation prompts
- Content generation prompts
- Prompt templates for marketing materials

## What Doesn't Belong Here

- Generated images (use `../img/`)
- Code or scripts
- Documentation prose
